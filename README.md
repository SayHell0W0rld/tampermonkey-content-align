# 网页内容对齐助手 (Content Align)

浏览器油猴脚本，解决网页左对齐导致久坐时身体不自觉左倾的问题。

## 功能

### 内容对齐（右下角按钮）

| 模式 | 说明 |
|------|------|
| ⬜ 原始 | 恢复默认 |
| ⬛ 居中 | 内容居中显示（扩大宽度 + margin auto）|
| ◎ 聚焦 | 鼠标所在内容区不变，其余磨砂玻璃虚化（blur + brightness + 半透明遮罩）|
| ⊙ 跟踪 | 悬停预览（蓝色高亮 + 虚化），点击锁定居中，双击解锁 |

### 阅读辅助（悬停主按钮展开，可叠加）

| 模式 | 说明 |
|------|------|
| 📖 关闭全部 | 关闭所有阅读辅助 |
| 🧠 仿生阅读 | 中文：每词首字加粗（font-weight:900）；英文：按词长分级加粗 + 渐隐 |
| 📏 阅读尺 | 半透明高亮条跟随鼠标，防止跳行 |
| 🎨 段落彩条 | 相邻段落交替柔和背景色，视觉分块 |
| 🌙 舒缓背景 | 米色护眼背景 + 文字颜色优化 |

## 安装

1. 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/)（Chrome/Edge/Firefox）
2. 打开 Tampermonkey 面板 → 新建脚本
3. 粘贴 `content-align.user.js` 内容 → 保存
4. 访问任意网页，右下角出现圆形按钮即生效

## 使用

### 交互方式
- **单击主按钮**（右下角）→ 弹出内容对齐菜单
- **悬停主按钮** → 下方浮现阅读辅助小按钮（📖）
- **单击阅读辅助按钮** → 弹出阅读辅助菜单
- **按钮自动隐藏** → 5秒不活动后按钮滑入右侧边栏，仅露出一半，鼠标靠近边缘自动弹出

### 快捷键
| 快捷键 | 功能 |
|--------|------|
| `Alt+1` | 原始模式 |
| `Alt+2` | 居中模式 |
| `Alt+3` | 聚焦模式 |
| `Alt+4` | 跟踪模式 |
| `Alt+Q` | 仿生阅读 开/关 |
| `Alt+W` | 阅读尺 开/关 |
| `Alt+E` | 段落彩条 开/关 |
| `Alt+R` | 舒缓背景 开/关 |
| `Alt+0` | 关闭所有阅读辅助 |

### 跟踪模式
鼠标移到内容区块 → 蓝色高亮预览 → **点击锁定** 居中 → **双击解锁**

### 与沉浸式翻译的关系
仿生阅读和沉浸式翻译**不能同时使用**。仿生阅读会修改DOM结构，导致翻译插件读取碎片化文本或重新翻译。如需使用翻译功能，请先关闭仿生阅读。

### 排除站点
油猴菜单 → 📋 管理排除站点（支持 `*.youtube.com` 通配符）

## 区块检测算法

脚本自动识别页面的"内容区块"（如文章正文、推荐列表容器），而非最小的 DOM 元素。

**判断标准**：父容器有 2-6 个显著子元素 → 这就是一个内容区块

| 父容器子元素数 | 含义 | 动作 |
|---|---|---|
| 1 个 | 页面容器层 | 继续向上 |
| 2-6 个 | 内容区块（main/sidebar/nav） | 返回 |
| 7+ 个 | 列表/网格内部 | 返回列表容器 |

效果：博客正文中鼠标移到图片上 → 返回整个 `<article>`，不会单独虚化图片。

## 排除站点

油猴菜单 → 📋 管理排除站点

支持通配符格式：`*.youtube.com`、`*.netflix.com`、`*.twitch.tv`

## 技术细节

- **按钮挂载到 `documentElement`**：不受 body `transform` 影响
- **菜单同样挂载到 `documentElement`**：右移模式下菜单不会跟着移走
- **`elementFromPoint` 查询前临时去掉 transform**：确保坐标准确
- **点击拦截器**：跟踪模式下虚化区域的 click 事件被 `preventDefault`，居中区块的链接正常可点
- **SPA 支持**：MutationObserver 监听 URL 变化，自动重新应用
